# 마동석 랜덤 사진 콜렉션 🚀

> 마동석 형님의 다양한 사진을 랜덤으로 볼 수 있는 크롬 익스텐션입니다.


# 주요기능

- 📷 **사진 랜덤으로 보기**: 랜덤사진 버튼 클릭시 마동석 형님의 사진을 랜덤으로 가져옵니다.

- 📌 **이미지 URL 제출**: 추가하고싶은 이미지 URL을 제출해주시면 디스코드에 전송되며, 전송된 이미지는 검토 후 추가합니다.
